Bài cuối kì sử dụng cơ sở dữ liệu mysql

Các bước cần chuẩn bị để chạy project:
	Import tất cả file sql trên xampp hoặc mysql workbench
	Vào file __init__.py trong folder app của từng serive tìm dòng code kết nối với database và thay đổi port cho phù hợp
	Chạy lệnh pip install -r requirements.txt để tải về các thư viện cần thiết
	Lần lượt nhấn lệnh python run.py cho từng service để khởi chạy
	Truy cập link http://127.0.0.1:5000 để truy cập trang web
